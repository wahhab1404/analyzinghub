### If You Lost Your .env File | إذا فقدت ملف .env

**English:**
1. Open `CREDENTIALS_QUICK_RESTORE.txt`
2. Copy all credentials
3. Create new `.env` file

2. انسخ جميع البيانات
3. أنشئ ملف `.env` جديد
4. الصق واحفظ
5. شغّل: `npm run verify:credentials` للتحقق
6. شغّل: `npm run dev`
