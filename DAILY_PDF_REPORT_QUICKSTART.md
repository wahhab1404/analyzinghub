# Daily PDF Report - Quick Start Guide

## What Was Created

A fully automated daily trading report system that:
- Generates beautiful HTML reports with trading statistics
- Sends professional summaries to all Telegram channels
- Runs automatically every weekday at 4 PM ET (market close)
- Stores historical reports in database

## Quick Test

Run this command to test immediately:
```bash
npm run test:pdf-report
```

### Report Statistics Included
1. **Total Trades** - All trades created today
2. **Active Trades** - Still open positions
3. **Closed Trades** - Completed today
4. **Expired Trades** - Expired options
5. **Average Profit** - Mean of all max profits
6. **Max Profit** - Highest profit achieved
7. **Win Rate** - Percentage of profitable trades

### HTML Report Features
- Professional gradient header with branding
- 7-stat summary grid with color coding
- Detailed trade cards showing:
  - Symbol & Option Type (CALL/PUT)
  - Strike Price
  - Entry Price
  - **Highest Price** (maximum reached)
  - Current Price
  - Max Profit %
  - Status & Exit Reason

## Automated Schedule

The system automatically runs:
- **Time**: 4:00 PM ET (21:00 UTC)
- **Days**: Monday - Friday
- **Trigger**: pg_cron job

No manual intervention needed!

## Configuration

## Troubleshooting

### Report Not Generating
1. Check if trades exist for today
2. Verify edge function is deployed
3. Check Supabase function logs

### Telegram Not Receiving
1. Verify `TELEGRAM_BOT_TOKEN` is set
2. Check analyzer plans have valid channel IDs
3. Test bot permissions in channels

### Wrong Statistics
1. Ensure `contract_high_since` is tracked
2. Verify `max_profit_percent` calculations
3. Check trade status values
